Три в степени

При вводе числа опредяет является число тройкой в n-ой степени

Мой код

const isPowerOfThree = (num)=>{
if (num===1){
return true;
}else if(num<1){
return false;
}
num=num/3;
return isPowerOfThree(num);
}
export default isPowerOfThree;

Код преподавателя

export default (num) => {
let current = 1;
while (current '<' num) {
current *= 3;}
return current === num;
}
